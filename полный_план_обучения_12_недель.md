
# Полный план обучения разработчика (12 недель)
Цель программы: развитие разработчика уровня Middle‑ до уверенного Middle+.

Программа разделена на три этапа:
1–3 недели — фундамент
4–9 недели — инструменты, инфраструктура и углубление
10–12 недели — production инженерия и архитектура

---

# Недели 1–3 — Фундамент

Фокус:
- TypeScript базовый и прикладной
- Архитектура Vue/Nuxt
- DevTools и анализ производительности
- Базовая работа с AI инструментами

Основные темы:

TypeScript
- utility types
- type guards
- discriminated unions
- strict mode
- типизация API контрактов

Nuxt
- структура проекта
- composables
- SSR
- жизненный цикл компонентов

DevTools
- Network
- Performance
- Lighthouse

Практика:
- рефакторинг Vue компонента
- типизация API клиента
- анализ производительности страницы

---

# Недели 4–6 — Архитектура и инфраструктура

Фокус:
- продвинутая архитектура Nuxt
- сборка проекта
- Docker и CI/CD

Основные темы:

Nuxt
- middleware
- runtime config
- error handling

Сборка
- Vite
- Rollup
- анализ bundle

Практика
- bundle анализ
- tree shaking
- code splitting

Infrastructure
- Docker multi stage build
- CI/CD pipeline

Фокус:

# Финал программы

После завершения программы разработчик должен уметь:

- проектировать Nuxt архитектуру
- анализировать производительность
- оптимизировать bundle
- работать с Docker и CI/CD
- интегрировать AI
- строить realtime системы
- анализировать production проблемы

Финальные активности:

1. архитектурное интервью
2. демонстрация финального проекта
3. доклад для команды
