Формат обучающей задачи

Каждая обучающая задача должна содержать следующие поля.

1. Тип задачи
   Название блока обучения.

2. Материалы
   Ссылки на документацию, статьи, курсы.

3. Продолжительность
   Время, выделенное на изучение.

4. Практическое задание
   Реальная задача внутри проекта.

5. Теоретическое задание (для интервью)
   Вопросы, которые тимлид задаёт разработчику.

6. Теоретическое задание (подготовить ответ)
   Тема, которую разработчик готовит к следующей встрече.
